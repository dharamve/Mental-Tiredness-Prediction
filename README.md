# Mental-Tiredness-Prediction
This project, Mental Tiredness Prediction, aims to build a Machine Learning model that predicts a person’s mental tiredness score based on various lifestyle, workplace, and behavioral factors. The dataset contains information related to screen time, number of decisions made, context switching, deep work duration, caffeine intake, sleep quality.

# Mental Tiredness Prediction

## Problem Statement
Predict mental tiredness score using daily habits, work patterns, and environmental factors.

## Objective
Build a machine learning model to predict mental tiredness levels.

## Dataset Information
- Total Records: 15,000
- Features: 17 input features + 1 target variable
- Target Variable: mental_tiredness_score
